## Что Может проект:

- **Три классических шифра:**
  - Шифр Цезаря (сдвиг букв)
  - Шифр Виженера (полиалфавитный шифр)
  - Шифр Атбаш (перевёрнутый алфавит)
- **Поддержка любого алфавита** через внешний файл
- **Полная проверка входных данных** с обработкой ошибок
- **Интерактивный консольный интерфейс**
- **Подробные тесты** для проверки корректности работы
- **Полная документация** в формате reST

## Требования для работы проекта:

- Python 3.6 или выше
- Файл с алфавитом в кодировке UTF-8
