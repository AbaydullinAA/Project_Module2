# Проект: Шифровальная станция

Консольное приложение для работы с тремя классическими шифрами: Цезаря, Виженера и Атбаш.

4. [Использование](#использование)
5. [Тестирование](#тестирование)
6. [Документация](#документация)

## Что Может проект:

- **Три классических шифра:**
  - Шифр Цезаря (сдвиг букв)
  - Шифр Виженера (полиалфавитный шифр)
  - Шифр Атбаш (перевёрнутый алфавит)
- **Поддержка любого алфавита** через внешний файл
- **Полная проверка входных данных** с обработкой ошибок
- **Интерактивный консольный интерфейс**
- **Подробные тесты** для проверки корректности работы
- **Полная документация** в формате reST

## Требования для работы проекта:

- Python 3.6 или выше
- Файл с алфавитом в кодировке UTF-8

## **Установка проекта**

Для запуска основной программы:

1. **Введите в консоль**
    python3 main.py

## Использование

1. Введите путь к файлу с алфавитом (например, alphabet.txt)
2. Выберите шифр из предложенных вариантов
3. Выберите операцию: шифрование или дешифрование
4. Введите ключ (если требуется для выбранного шифра)
5. Введите текст для обработки
6. Получите результат в консоли

## Тестирование

**Тесты проверяют:**

- Корректность чтения алфавита
- Валидацию входных данных
- Работу всех трёх шифров
- Обработку ошибок

## Документация
 **Для получения Документации в виде HTML страницы:**

1. Установите PyDoctor
2. pydoctor --make-html --html-output=docs_all --project-base-dir="." --docformat=restructuredtext ciphers.py main.py test_ciphers.py
3. Ввести в консоль < open docs_all/index.html >
